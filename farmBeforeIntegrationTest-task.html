farmBeforeIntegrationTest task

Gretty automatically invokes this task before farmIntegrationTest task.

Warning
 Internal task, please, don’t invoke it on command line!

Effects

  1. Embedded jetty starts in separate java process against the web-apps of given farm and listens for HTTP-requests on port (default 8080).

  2. Gradle script waits until jetty is online.

  3. Gradle script proceeds to integration test task.

See also: Farm integration tests.
